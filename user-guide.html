What is the Tuner?

The Tuner is a compositional environment for creating evolving musical landscapes. It's a controlled musical engine that lets you define musical boundaries and behaviors - then the system composes in real-time within those boundaries, creating musical compositions that that follow your directions, but that never repeat exactly the same way twice.

Think of it as conducting an intelligent orchestra: you set the rules for how each musician plays, and they improvise together, creating organic landscapes that breathe and evolve.

Core Philosophy: Controlled Randomness

The Tuner's innovation is the behavior slider - found on every parameter. This single control transforms static settings into living, breathing music:

  • Low behavior (0-25%): Parameters change slowly and smoothly, creating calm, meditative soundscapes
  • Medium behavior (26-75%): Balanced evolution with both gentle and surprising moments
  • High behavior (76-100%): Dramatic, unpredictable changes that explore the full range you've defined

You define the boundaries (minimum and maximum values). The behavior slider controls how the system moves within those boundaries.

What is a Voice?

The Tuner offers up to 16 independent voices, each with its own complete set of parameters. Each voice is like a musician in an ensemble:

  • One voice might play slow, deep bass notes that drone for minutes
  • Another might play rapid, high-pitched patterns that flutter and dance
  • A third might enter only at specific times, adding occasional punctuation
  • Together, they weave a complex, evolving musical tapestry

Each voice has 20 adjustable parameters organized into five loose categories:

üéµ Instrument & Sound

  • Which sound/instrument to use
  • Which notes are available (full 88-key piano range)
  • How many notes play at once (1-16 simultaneous notes)
  • Attack intensity (soft to aggressive)
  • Pitch detuning effects
  • Portamento (note-to-note gliding)

üéöÔ∏è Mixing & Levels

  • Volume range
  • Stereo positioning (left-center-right)

üéØ Rhythm & Timing

  • Tempo (each voice can have its own speed!)
  • Which note durations are available (thirty-second notes through four whole notes)
  • Rest patterns (silence between notes)
  • Life Span (when the voice enters, exits, and whether it repeats)

üåÄ Modulation Effects

  • Tremolo (volume pulsing)
  • Chorus (thickening and shimmer)
  • Phaser (sweeping frequency effect)

üèõÔ∏è Spatial Effects

Professional Audio Standards

The Tuner adheres to professional music production standards:

  • Full MIDI range: Complete 88-key piano keyboard (A0 to C8) with proper note naming
  • Standard musical notation: All rhythm values follow traditional music theory (triplets, dotted notes, etc.)
  • Velocity-sensitive playback: Notes respond dynamically to attack settings, with realistic envelopes
  • Professional effects: Studio-quality reverb, delay, chorus, phaser, and tremolo with precise control
  • Polyphonic chord generation: Intelligent algorithm creates musically pleasing harmonies based on music theory

Quick Start: Your First Composition in 5 Minutes

Let's create a simple two-voice composition to understand the basics:

Step 1: Set Up Voice 1 (Deep Bass Drone)

  1. Click the Voice 1 tab (left sidebar) - it's enabled by default
  2. Expand the Instrument rollup
    • Select an instrument - choose "Acoustic Bass" or any bass sound from the dropdown
  3. Expand the Melodic Range rollup:
    • Click notes in the lowest two octaves of the piano keyboard (left side)
    • Or click-and-drag across a range of low notes
    • Set behavior to 25% (smooth, connected movement)
  4. Expand "Rhythm" rollup:
    • Check only "Whole Note" and "Two Whole Notes" (very slow)
    • Set Rhythms behavior to 50%
    • Check only "No Rests" (plays continuously)
  5. Expand "Volume" rollup:
    • Set Volume from 30% to 60% with behavior at 75% (gentle breathing)
  6. Expand "Stereo Balance" rollup:
    • Set Stereo Balance from -50 to +50 with behavior at 80% (wanders in stereo field)

Step 2: Set Up Voice 2 (High Sparkle)

  1. Click the Voice 2 tab and check the checkbox to enable it
  2. Expand the Instrument rollup
    • Select "Vibraphone" or "Music Box"
    • Click notes in the highest two octaves of the piano keyboard (right side)
    • Set behavior to 100% (maximum variation)
  3. Expand "Rhythms" rollup:
    • Check "Sixteenth Notes", "Eighth Notes", and "Quarter Notes" (fast patterns)
    • Set Rhythms behavior to 75%
  4. Expand "Rests" rollup:
    • Check "Eighth Notes" and "Quarter Notes" for Rests (breathing space)
    • Set Rests behavior to 60%
  5. Expand "Volume" rollup:
    • Set Volume from 20% to 45% with behavior at 85% (fluttering dynamics)
  6. Expand "Stereo Balance" rollup:
    • Set Stereo Balance from -100 to +100 with behavior at 90% (dramatic panning)

Step 3: Listen

  1. On Voice 1 tab, press the PREVIEW button to hear it alone. Press STOP when ready
  2. On Voice 2 tab, press the PREVIEW button to hear it alone. Press STOP when ready
  3. Press the PLAY button (top bar) to hear both voices together
  4. Adjust any parameter in real-time - changes take effect immediately
  5. Watch the evolution - the music will never sound exactly the same way twice

How Parameters Work: The Details

Each parameter follows the same logic: you set boundaries (min/max), and the behavior slider controls how the system moves within those boundaries during playback.

The Behavior Slider Formula

Every parameter has a behavior slider (except dropdown menus). Here's what behavior percentages mean:

  • 0%: Parameter stays at the exact midpoint between min and max (completely static)
  • 1-25%: Smooth, gentle changes - values drift slowly
  • 26-50%: Moderate evolution - noticeable but musical changes
  • 51-75%: Active variation - frequent, substantial changes
  • 76-100%: Maximum drama - wild jumps within your defined boundaries

Example: Volume set from 20% to 80% with 50% behavior

  • System starts near the middle (around 50% volume)
  • Gradually drifts up and down within the 20-80% range
  • Never goes below 20% or above 80%
  • Changes are moderate and musical

Example: Volume set from 20% to 80% with 100% behavior

  • System makes dramatic leaps
  • Might jump from 25% to 75% suddenly
  • Still respects your 20-80% boundaries
  • Creates dynamic, unpredictable energy

Detailed Parameter Reference

üéµ INSTRUMENT

Purpose: Choose the sound source for this voice

Control: Simple dropdown menu

  • Lists all sounds available in your system
  • Default sounds use your computer's built-in synthesizer
  • With Professional Sample Library (free account): access to authentic instrument recordings

No behavior slider - sound selection is fixed per voice

üéµ MELODIC RANGE

Purpose: Define which notes the voice can play

Control: Interactive 88-key piano keyboard (A0 to C8)

  • Click individual keys to select/deselect notes
  • Click-drag to select ranges
  • Selected notes highlighted in coral/orange
  • Info display shows your selection (e.g., "A2 to C5 (18 notes)")

Musical Intelligence: The system doesn't just pick random notes from your selection. It uses a chord generation algorithm that understands music theory:

  • Creates musically pleasing harmonies (major, minor, diminished, augmented chords)
  • Chooses intervals that sound good together
  • Adapts chord complexity based on your Polyphony setting

Behavior Slider controls melodic motion:

  • Low (0-25%): Stepwise motion, notes close together, smooth melodies
  • Medium (26-75%): Mix of steps and leaps, balanced movement
  • High (76-100%): Wide intervalic leaps, exploring full range dramatically

Tip: For bass voices, select only low notes. For sparkle/color, select only high notes. For full-range instruments, select broadly and let the system explore.

üéµ PHRASE STYLES

Purpose: Control how notes are organized into melodic phrases - the shapes and patterns your melodies follow

Controls: Six melodic patterns plus optional breathing space

Each pattern has:

  • Checkbox: Enable/disable the pattern
  • Length slider: Set phrase length (2 to 24 notes)

The Six Melodic Patterns

1. Random (enabled by default)

  • Description: Each note is independently and unpredictably selected from your melodic range
  • Sound: Unpredictable, scattered, no melodic pattern
  • Use for: Atonal textures, percussive elements, chaotic energy, experimental sounds
  • Example: Length 2 = pure randomness, Length 12 = longer random sequences

2. Ascending

  • Description: Notes move upward through your melodic range, starting from a random position each phrase
  • Sound: Rising sequences, building energy, climbing motion
  • Behavior effect: 0% = smooth scalar steps (C‚ÜíD‚ÜíE‚ÜíF), 100% = large intervallic leaps (C‚ÜíG‚ÜíD‚ÜíB) while still ascending
  • Use for: Building intensity, uplifting passages, rising arpeggios

3. Descending

  • Description: Notes move downward through your range, starting from random positions
  • Sound: Falling sequences, releasing tension, downward motion
  • Behavior effect: 0% = smooth steps down, 100% = dramatic downward leaps
  • Use for: Calming passages, resolution, falling melodic lines

4. Pendulum

  • Description: Notes oscillate up to the top of your range, then down to the bottom, repeating
  • Sound: Wave-like back-and-forth motion, symmetrical arcs
  • Behavior effect: 0% = gentle swaying (small steps), 100% = dramatic swings (large leaps)
  • Use for: Rocking motion, cyclical patterns, balanced movement

5. Wave

  • Description: Notes follow a sine wave pattern through your range, creating smooth, flowing curves
  • Sound: Organic, flowing melodies with natural contours
  • Behavior effect: 0% = smooth flowing curves (many gentle waves), 100% = jagged angular motion (fewer dramatic waves)
  • Use for: Lyrical melodies, natural-sounding phrases, ambient textures

6. Spiral

  • Description: Starts near a random center point and gradually expands outward, alternating above and below
  • Sound: Growing intensity, expanding range, blooming outward
  • Behavior effect: 0% = gradual gentle expansion, 100% = rapid dramatic expansion with added chaos
  • Use for: Building complexity, expanding musical ideas, organic growth

Use Space Between Phrases (Breathe)

Purpose: Insert periods of complete silence between melodic phrases - separate from the RESTS parameter

Controls: Checkbox + Length slider (located in header row of Phrase Styles box)

  • Checkbox unchecked: No breathing space (continuous melodic phrases)
  • Checkbox checked: System probabilistically inserts silence gaps between phrases
  • Length slider: Sets duration of silence (2-24 rhythm units)

How Breathe Frequency Works: Controlled by the Behavior slider

  • 0% Behavior: Breathe rarely (~10% probability) - occasional gaps
  • 50% Behavior: Breathe sometimes (~50% probability) - frequent spaces
  • 100% Behavior: Breathe often (~90% probability) - gaps after almost every phrase

Important: Breathe is completely separate from the RESTS parameter

  • RESTS: Silence between individual notes (micro-level spacing)
  • BREATHE: Silence between entire phrases (macro-level phrasing)

Use for: Creating natural phrasing, adding dramatic pauses, reducing density, mimicking human breathing in performance

How Pattern Selection Works

Multiple Patterns Enabled: System randomly chooses from your enabled patterns

  • Each pattern uses its own length setting
  • Example: Ascending (length 6) + Wave (length 14) = alternates between 6-note ascending phrases and 14-note wave phrases

Behavior Slider Effect on Pattern Selection:

  • 0-50% Behavior: History avoidance - system tries to avoid repeating the same pattern consecutively
  • 50-100% Behavior: Pure random selection - any pattern can follow any other pattern

Single Pattern Enabled: System uses only that pattern (Behavior controls interval sizes within the pattern)

No Patterns Enabled: System automatically enables Random pattern (length 2) and continues playing

The Behavior Slider - Dual Purpose

The Behavior slider for PHRASE STYLES serves two musical functions simultaneously:

1. Pattern Switching Frequency: How the system chooses between enabled patterns

  • 0% Behavior: Maximum pattern variety - rarely repeats same pattern
  • 100% Behavior: Pure randomness - any pattern can repeat multiple times

2. Interval Sizes Within Patterns: How far notes jump within a pattern

  • 0% Behavior: Small steps (adjacent notes, scalar motion)
  • 50% Behavior: Mix of steps and leaps
  • 100% Behavior: Large intervallic jumps (dramatic, wide-ranging motion)

Practical Examples

Example 1 - Smooth Ambient Pad:

Enabled: Wave only
Wave Length: 16 notes
Breathe: Unchecked
Behavior: 25%

Result: Continuous flowing melodies with gentle curves, smooth scalar motion (perfect for background pads)

Example 2 - Dramatic Lead Voice:

Enabled: Ascending + Descending + Pendulum
All Lengths: 8 notes
Breathe: Checked, Length 6
Behavior: 85%

Result: Dramatic phrases with large leaps, frequently switching between ascending/descending/pendulum patterns, with frequent silence gaps for breathing (perfect for expressive solo lines)

Example 3 - Textural Variety:

Enabled: Random (length 2) + Wave (length 18)
Breathe: Checked, Length 4
Behavior: 60%

Result: Alternates between short chaotic bursts (2-note random) and long flowing phrases (18-note wave), with occasional breathing space - creates compelling contrast

Understanding Phrase Length

What is a Phrase? A phrase is a complete musical gesture - a sequence of notes that forms a coherent melodic statement

  • Short phrases (2-6 notes): Quick, repetitive, rhythmic focus
  • Medium phrases (7-12 notes): Balanced, natural phrasing
  • Long phrases (13-24 notes): Elaborate, developmental, melodic focus

Each Pattern Can Have Different Lengths:

  • Random can be set to 3 notes (short bursts)
  • Wave can be set to 20 notes (long flowing lines)
  • System will alternate: short chaotic bursts, then long flowing phrases

Random Starting Notes

Key Feature: Each phrase starts from a different random position in your melodic range

  • An 8-note Ascending phrase doesn't always start on C - it might start on E, or G, or any note in your range
  • This ensures the same pattern sounds fresh each time it appears
  • Phrases wrap around when they reach the edge of your range

Common Pattern Combinations

For Ambient/Drone: Wave only, length 18-24, Behavior 20%

For Minimalism: Ascending + Descending, length 4-6, Behavior 10%, Breathe enabled (length 8)

For Experimental/Atonal: Random only, length 2-4, Behavior 100%

For Lyrical Melody: Wave + Pendulum, length 10-14, Behavior 35%, Breathe enabled (length 4-6)

For Maximum Variety: Enable all 6 patterns with varying lengths, Behavior 75%

Advanced: Per-Voice Pattern Design

Since each voice has independent Phrase Styles, create complementary patterns across voices:

  • Voice 1 (bass): Ascending only, length 24, Behavior 10% (long, consistent upward motion)
  • Voice 2 (harmony): Wave only, length 12, Behavior 40% (flowing counterpoint)
  • Voice 3 (accent): Random only, length 2, Breathe enabled (length 12), Behavior 90% (sparse punctuation with lots of space)

Result: Three distinct melodic characters working together in complementary roles

Pattern Selection Strategy

When multiple patterns are enabled, the system selects randomly from your enabled set:

  • Low Behavior (0-50%): System tracks the last 2-3 patterns used and tries to avoid immediate repetition - creates variety
  • High Behavior (50-100%): Pure random selection - same pattern can appear multiple times in a row

Musical Insight: At low Behavior, enabling many patterns ensures maximum variety. At high Behavior, the system explores repetition and obsessive patterns.

Tips for Phrase Styles

  • Start with one pattern: Enable only Random or Ascending initially to understand how phrase length affects the music
  • Combine contrasting patterns: Ascending + Descending creates clear directional contrast
  • Use Breathe sparingly: At high Behavior (100%), breathing happens ~90% of the time - consider using 30-60% Behavior for more balanced phrasing
  • Match phrase length to rhythm: Shorter phrases (4-8) work well with fast rhythms (Sixteenth, Eighth Notes), longer phrases (12-20) suit slower rhythms
  • Wide melodic ranges: Use longer phrases (12+) to fully explore the range - short phrases (2-4) in wide ranges create more scattered effects
  • Experiment with Behavior: The same pattern sounds completely different at 0% vs 100% - try extreme settings to hear the full range

üéµ POLYPHONY

Purpose: How many simultaneous notes can play at once

Control: Dual-handle slider (1 to 16 notes)

  • 1 note: Monophonic - single melody line
  • 2-4 notes: Small chords, duets, simple harmonies
  • 5-8 notes: Rich chords, complex harmonies
  • 9-16 notes: Dense clusters, orchestral textures

Behavior Slider controls how polyphony changes:

  • 0%: Fixed note count (both handles same position)
  • Low: Gradual shifts (1 note, then 2, then 1, then 3...)
  • High: Dramatic changes (suddenly 1 note, then 12, then 2...)

Musical Intelligence: The system's chord generation algorithm adapts to your polyphony setting:

  • At 1-2 notes: Creates melodic intervals
  • At 3-4 notes: Builds triads and seventh chords
  • At 5-8 notes: Adds extensions (9ths, 11ths, 13ths)
  • At 9+ notes: Creates cluster chords and complex voicings

Tip: Bass voices work well with 1-2 notes. Pad/background voices sound lush with 4-8 notes.

üéµ ATTACK VELOCITY

Purpose: Controls how hard notes are "struck" - affects brightness and intensity

Control: Dual-handle slider (0 to 127, MIDI standard)

  • 0-40: Soft, gentle, mellow tones
  • 41-80: Medium dynamics, balanced
  • 81-127: Loud, bright, aggressive attack

Real-time Effect:

  • Waveform selection: Soft attacks use sine waves (mellow), hard attacks use square waves (bright/harsh)
  • Filter cutoff: Soft attacks roll off high frequencies, hard attacks open the filter (more brightness)
  • Envelope shape: Soft attacks have slower attack time, hard attacks have snappy, immediate onset

Behavior Slider controls velocity changes:

  • 0%: Fixed dynamics
  • Low: Gentle breathing (like a musician's natural phrasing)
  • High: Dramatic dynamic contrasts (whisper to scream)

Tip: For ambient pads, use 20-50 with low behavior. For percussive parts, use 80-127 with high behavior.

üéµ DETUNING

Purpose: Shift pitch slightly sharp or flat (measured in cents: 100 cents = 1 semitone)

Control: Dual-handle slider (-50 to +50 cents)

  • 0 cents: Perfect tuning
  • -50 to -10 cents: Slightly flat (darker, warmer)
  • +10 to +50 cents: Slightly sharp (brighter, tenser)

Musical Effects:

  • Slight detuning (¬±5-15 cents): Creates natural chorusing, thickness
  • Moderate detuning (¬±15-30 cents): Quarter-tone effects, tension
  • Heavy detuning (¬±30-50 cents): Dissonant, experimental sounds

Behavior Slider adds micro-detuning evolution: Even during sustained notes, the pitch subtly wobbles creating organic, living tones

To disable: Set both handles to 0

Tip: Try ¬±8 cents with 40% behavior for lush, slightly out-of-tune ensemble effect (like vintage synthesizers)

üéµ PORTAMENTO GLIDE TIME

Purpose: Smooth pitch gliding between notes (like a slide on trombone or violin)

Control: Dual-handle slider (0 to 1000 milliseconds)

  • 0 ms: No glide, immediate pitch change
  • 50-200 ms: Quick, subtle glide
  • 200-500 ms: Noticeable glide
  • 500-1000 ms: Dramatic, slow glide

Behavior Slider controls glide time variation:

  • Sometimes quick snaps between notes
  • Sometimes slow, expressive slides

To disable: Set both handles to 0

Tip: Sounds great on bass voices (150-400ms) and lead melodies (100-250ms). Usually too much on fast, busy voices.

Control: Dual-handle slider

  • 0%: Silent
  • 25-50%: Background/pad level
  • 50-75%: Present but not dominant
  • 75-100%: Prominent/lead level

Behavior Slider creates dynamic breathing:

  • Low: Gentle swells and dips (natural phrasing)
  • High: Dramatic volume changes (sudden appearances and disappearances)

Real-time updates: Changes apply immediately during playback

Tip: For evolving textures, try 20-60% with 85% behavior - voices fade in and out organically

üéöÔ∏è STEREO BALANCE

üéØ TEMPO (BPM)

Purpose: Control playback speed (40 to 240 beats per minute)

Control: Dual-handle slider

  • 40-60 BPM: Very slow, meditative
  • 60-90 BPM: Slow, relaxed
  • 90-120 BPM: Moderate, walking pace
  • 120-160 BPM: Upbeat, energetic
  • 160-240 BPM: Very fast, intense

Revolutionary Feature: Each voice has independent tempo

  • Voice 1 might play at 60 BPM (slow bass)
  • Voice 2 might play at 180 BPM (fast sparkle)
  • Creates polyrhythmic textures where different speeds interact

Behavior Slider controls tempo changes:

  • 0%: Fixed tempo
  • Low: Gradual tempo drifts (rubato feel)
  • High: Dramatic speed changes

SYNC Button: Copy this voice's tempo settings (min, max, behavior) to all other voices instantly - useful when you want synchronized ensemble playing

Tip: Voices at very different tempos (60 BPM vs 180 BPM) create fascinating rhythmic interactions

üéØ RHYTHMS

Purpose: Define which note durations are available

Control: Checkbox grid with 15 options

  • Thirty-second Notes (very fast)
  • Thirty-second Note Triplets
  • Sixteenth Notes
  • Sixteenth Note Triplets
  • Eighth Notes
  • Eighth Note Triplets
  • Quarter Note Triplets
  • Quarter Notes (standard beat)
  • Half Note Triplets
  • Half Notes
  • Whole Note Triplets
  • Whole Note (4 beats)
  • Two Whole Notes (8 beats)
  • Three Whole Notes (12 beats)
  • Four Whole Notes (16 beats)

How it Works:

  • Check multiple boxes: System randomly selects from your palette
  • Check one box: Fixed rhythmic value
  • Selected boxes highlighted in blue

Behavior Slider:

  • 0%: Always uses first checked rhythm (no variation)
  • 1-100%: Random selection from all checked rhythms (higher = more frequent changes)

Tip: For bass, check only slow values (Half Notes, Whole Notes). For percussion/sparkle, check fast values (Sixteenth, Eighth Notes). For melody, mix several adjacent values.

üéØ RESTS

Purpose: Define silence durations between notes

Control: Checkbox grid (same 15 durations as Rhythms, plus special "No Rests" option)

Special "No Rests" Checkbox:

  • When checked: Voice plays continuously with no pauses (unchecks all other rest values automatically)
  • When unchecked: Voice uses selected rest values for spacing

Auto-Exclusive Logic:

  • Checking "No Rests" unchecks all other rest values
  • Checking any other rest value unchecks "No Rests"

Behavior Slider:

  • 0%: Fixed rest duration (if multiple checked, always uses first one)
  • 1-100%: Random selection creates varied spacing patterns

Tip: "No Rests" works great for drones and pads. For rhythmic voices, check 2-3 rest values that complement your chosen rhythm values.

üéØ LIFE SPAN

Purpose: Control precisely when and how long a voice plays - creates entrances, exits, and layering

Controls:

1. Total Time Length (text input, MM:SS format)

  • Range: 0:05 (5 seconds) minimum to 60:00 (1 hour) maximum
  • Defines the timeline for all entrance/exit timing

2. Beat Unit (dropdown menu)

  • Choose how time is displayed in tooltips
  • Options: All 15 rhythm values (Thirty-second Notes through Four Whole Notes)
  • Example: Set to "Quarter Notes" - tooltips show "120 beats (2:00)"

3. Entrance 1, 2, 3 (three dual-handle sliders)

  • Left handle: When voice starts playing (Enter time)
  • Right handle: When voice stops playing (Exit time)
  • Set to infinity (‚àû): Voice plays until manually stopped
  • Set both to 0:00: Disable this entrance slot

4. Repeat (checkbox)

  • Unchecked: Pattern plays once then stops permanently
  • Checked: Pattern cycles continuously based on Total Time Length

Tooltip Display: Each slider handle shows TWO pieces of information:

  • Musical time: "120 beats" (based on your Beat Unit selection)
  • Clock time: "(2:00)" in minutes:seconds

How Multiple Entrances Work:

  • Voice plays during ANY entrance time window
  • Entrance windows can overlap
  • If times overlap, voice plays continuously through the overlap

How Repeat Works:

  • When checked, entire pattern cycles based on Total Time Length
  • Example: Total Time 1:00, Entrance 1 from 0:00-0:20
  • Pattern: Play 20 seconds, silent 40 seconds, repeat forever
  • When unchecked, voice stops permanently after last exit

Important: Life Span timing is independent of Tempo

  • Life Span times are absolute clock time
  • Tempo only affects how fast notes play during active periods

Example 1 - Delayed Entrance (Classical Orchestration):

Total Time Length: 5:00
Entrance 1: 1:00 to ‚àû (infinity)
Repeat: Unchecked

Result: Silence for 1 minute, then voice enters and plays continuously for remaining 4 minutes (perfect for building intensity)

Example 2 - Multiple Entrances (Call and Response):

Total Time Length: 3:00
Entrance 1: 0:00 to 0:30
Entrance 2: 1:00 to 1:30
Entrance 3: 2:00 to ‚àû
Repeat: Unchecked

Result: Voice plays three times - first 30 seconds, again at 1:00-1:30, finally from 2:00 onwards (creates rhythmic punctuation)

Example 3 - Repeating Pattern (Minimalist Texture):

Total Time Length: 1:00
Entrance 1: 0:00 to 0:20
Entrance 2: 0:40 to 1:00
Repeat: Checked

Result: Voice plays seconds 0-20 and 40-60 of each minute, cycling forever (creates pulsing in-and-out pattern)

Example 4 - Overlapping Entrances (Dense Layering):

Total Time Length: 2:00
Entrance 1: 0:00 to 1:00
Entrance 2: 0:30 to 1:30
Entrance 3: 1:00 to 2:00
Repeat: Unchecked

Result: Continuous sound from 0:00-2:00, but different entrance "layers" activate and deactivate creating subtle textural shifts

Tips:

  • Use delayed entrances to build complexity gradually
  • Set some voices to short entrances (punctuation/accents)
  • Use Repeat with short patterns (5-15 seconds) for pulsing textures
  • Combine voices with different Life Span patterns for complex orchestration
  • Set Total Time Length to your desired composition duration, or use infinity (‚àû) for open-ended exploration

üåÄ TREMOLO

Purpose: Rhythmic volume pulsing (classic "wobble" effect)

Controls: Two dual-handle sliders

1. Speed (0.5 Hz to 12 Hz)

  • 0.5-2 Hz: Slow, gentle pulsing
  • 2-5 Hz: Medium wobble (classic tremolo)
  • 5-12 Hz: Fast vibrato-like effect

2. Depth (0% to 100%)

  • 0%: No effect (disabled)
  • 25-50%: Subtle pulsing
  • 50-75%: Noticeable wobble
  • 75-100%: Dramatic on/off pulsing

Behavior Slider: Controls how Speed and Depth evolve

  • 0%: Fixed tremolo
  • Low-High: Tremolo intensity varies during playback

To disable: Set both Depth handles to 0%

Tip: Try 3-5 Hz Speed with 40-60% Depth for classic tremolo. Higher speeds (8-12 Hz) create vibrato-like effects.

üåÄ CHORUS

Purpose: Thickening and shimmer effect (like multiple instruments playing slightly out of tune)

Controls: Two dual-handle sliders

1. Speed (0.2 Hz to 3 Hz)

  • 0.2-0.8 Hz: Slow, wide modulation
  • 0.8-1.5 Hz: Classic chorus speed
  • 1.5-3 Hz: Fast, vibrato-like

2. Depth (0% to 100%)

  • 0%: No effect (disabled)
  • 20-40%: Subtle thickening
  • 40-70%: Rich, lush chorus
  • 70-100%: Extreme detuning

Technical: Uses three delay lines with different modulation rates for authentic analog chorus sound

Behavior Slider: Controls Speed and Depth variation

To disable: Set both Depth handles to 0%

Tip: 1.0 Hz Speed with 45% Depth creates lush, professional chorus. Great on pads and strings.

üåÄ PHASER

Purpose: Sweeping, swirling effect (frequency notches moving through spectrum)

Controls: Two dual-handle sliders

1. Speed (0.1 Hz to 2 Hz)

  • 0.1-0.5 Hz: Slow sweep (spacey)
  • 0.5-1.0 Hz: Medium sweep (classic phaser)
  • 1.0-2.0 Hz: Fast sweep (intense)

2. Depth (0% to 100%)

  • 0%: No effect (disabled)
  • 20-40%: Subtle movement
  • 40-70%: Strong phasing
  • 70-100%: Extreme sweep

Technical: Uses four cascaded all-pass filter stages (authentic analog phaser design)

Behavior Slider: Controls Speed and Depth variation

To disable: Set both Depth handles to 0%

Tip: 0.4 Hz Speed with 55% Depth creates classic '70s phaser sound. Combine with Chorus for complex modulation.

üèõÔ∏è REVERB

Purpose: Add spatial depth (simulates acoustic spaces from small rooms to cathedrals)

Controls: Two dual-handle sliders

1. Time (0.5 to 6.0 seconds)

  • 0.5-1.5s: Small room, tight ambience
  • 1.5-3.0s: Hall, medium space
  • 3.0-6.0s: Cathedral, vast space

2. Mix (0% to 100% wet/dry)

  • 0%: No reverb (completely dry)
  • 20-40%: Subtle ambience (natural)
  • 50-70%: Prominent space
  • 80-100%: Dominated by reverb (ethereal)

Behavior Slider: Both Time and Mix evolve during playback

  • 0%: Static reverb
  • Low: Gentle space changes
  • High: Dramatic acoustic shifts (small room suddenly becomes cathedral)

To disable: Set both Mix handles to 0%, or both Time handles to 0.5s

Tip: For ambient music, try 3.0-5.0s Time with 45-65% Mix and 60% behavior - creates evolving, breathing spaces

üèõÔ∏è DELAY

Purpose: Rhythmic echoes synchronized to tempo (creates depth and rhythmic complexity)

Controls: Three dual-handle sliders

1. Time (100ms to 2000ms)

  • Tooltips show both milliseconds and musical values
  • Example: "250ms = 1/8 (Eighth Note)" at 120 BPM
  • Automatically syncs to voice's current tempo
  • 100-300ms: Fast, tight echoes
  • 300-800ms: Medium delays (rhythmic punctuation)
  • 800-2000ms: Long delays (ambient wash)

2. Mix (0% to 100% wet/dry)

  • 0%: No delay (completely dry)
  • 20-40%: Subtle echo (adds depth)
  • 50-80%: Prominent repeats
  • 80-100%: Delay dominates

3. Feedback (0% to 100%)

  • 0%: Single echo
  • 30-50%: 2-3 repeats
  • 50-70%: 4-6 repeats
  • 70-90%: Long tail of fading echoes
  • 90-100%: Very long sustain (caution: can build up!)

Musical Synchronization:

  • Delay time stays locked to musical subdivisions as tempo changes
  • Example: Set to "1/4 note" - if tempo speeds up, delay time shortens automatically to maintain quarter-note spacing

Behavior Slider: All three parameters (Time, Mix, Feedback) evolve

  • 0%: Static delay
  • Low-High: Creates shifting, evolving echo patterns

To disable: Set both Mix handles to 0%

Example 1 - Rhythmic Eighth-Note Delays:

Time: 250ms-500ms (displays as "1/8" to "1/4" at 120 BPM)
Mix: 30%-50%
Feedback: 40%-60%
Behavior: 60%

Result: Rhythmically synchronized echoes that dance between eighth and quarter note timing with 3-5 repeats

Example 2 - Ambient Wash:

Time: 800ms-1500ms
Mix: 50%-70%
Feedback: 70%-85%
Behavior: 40%

Result: Long, overlapping delays create lush ambient texture with slowly evolving echo patterns

Caution: High feedback (85%+) with long delay times can create building, sustained echoes. This can be beautiful for ambient music but may overpower the mix. The system automatically limits maximum feedback to prevent runaway buildup.

Advanced Techniques

Polyrhythmic Textures

Each voice has independent tempo - use this to create complex interactions:

Example - Three-Speed Texture:

  • Voice 1 (bass): 60 BPM, Whole Notes (very slow foundation)
  • Voice 2 (mid): 120 BPM, Quarter Notes (walking pace)
  • Voice 3 (high): 180 BPM, Sixteenth Notes (rapid sparkle)

Result: Three different time scales interact, creating organic complexity

Layered Entrances

Use Life Span to build complexity gradually:

Example - 5-Minute Build:

Total Time Length: 5:00

Voice 1 (drone): 0:00 to ‚àû
Voice 2 (bass): 1:00 to ‚àû
Voice 3 (mid): 2:00 to ‚àû
Voice 4 (high): 3:00 to 4:00 (temporary appearance)
Voice 5 (lead): 3:30 to ‚àû

Result: Composition builds from solo drone through full texture, with temporary voice creating moment of climax

Call and Response

Use Life Span with non-overlapping time windows:

Example - Dialog:

Total Time Length: 2:00, Repeat: Checked

Voice 1: 0:00-0:15, 0:30-0:45, 1:00-1:15, 1:30-1:45
Voice 2: 0:15-0:30, 0:45-1:00, 1:15-1:30, 1:45-2:00

Result: Voices alternate in call-and-response pattern, never playing simultaneously

Stereo Choreography

Use extreme stereo settings with high behavior to create spatial movement:

Example - Swirling Stereo:

  • Voice 1: Balance -100 to -50, Behavior 85% (active on left)
  • Voice 2: Balance +50 to +100, Behavior 85% (active on right)
  • Voice 3: Balance -100 to +100, Behavior 95% (flies across entire field)

Result: Three-dimensional soundscape with voices moving independently

Evolution From Simple to Complex

Start with constrained settings, then expand:

First 2 Minutes:

  • Melodic Range: Only 5 notes selected
  • Rhythms: Only Quarter Notes checked
  • Polyphony: 1-2 notes
  • All behavior sliders: 20-30%

Last 3 Minutes:

  • Melodic Range: Full octaves selected
  • Rhythms: Multiple values checked
  • Polyphony: 4-8 notes
  • All behavior sliders: 70-85%

You can use two voices to do this and save it as a .json file using the Save button, or you can use it as a performance tool by manually adjusting parameters during playback, creating a live performance

Tips for Success

Start Simple

  • Enable only 1-2 voices initially
  • Use low behavior settings (20-40%) while learning
  • Stick to one or two octaves in Melodic Range
  • Check only 2-3 rhythm values

Build Gradually

  • Add voices one at a time
  • Preview each voice individually before enabling in full mix
  • Start with long, slow values (Whole Notes) and add faster elements later
  • Begin with low polyphony (1-2 notes) and increase if desired

Create Contrast

  • Mix slow and fast voices (different tempos)
  • Mix high and low (different melodic ranges)
  • Mix busy and sparse (different rhythm densities)
  • Mix loud and soft (different volume ranges)

Use Effects Subtly

  • Start with Mix/Depth at 20-30% and increase gradually
  • Too much reverb makes everything muddy
  • Too much delay creates rhythmic confusion
  • Less is often more with modulation effects

Embrace Evolution

  • Let the composition run for several minutes
  • Don't expect immediate perfection
  • The music should surprise you
  • Make small adjustments and observe results over time

Save Variations

  • Save multiple versions as you experiment
  • "Ambient-v1.json", "Ambient-v2.json", etc.
  • Easy to return to earlier versions
  • Compare different approaches

Use PREVIEW Extensively

  • Test each voice in isolation
  • Easier to hear what each voice contributes
  • Catch problems before adding to full mix
  • Adjust parameters without stopping main playback

Think in Layers

  • Foundation layer: Bass, drones (Voices 1-3)
  • Middle layer: Harmony, texture (Voices 4-8)
  • Top layer: Melody, sparkle (Voices 9-12)
  • Accent layer: Punctuation, effects (Voices 13-16)
